It's crucial to select the proper project if you have a few



Scroll down and click on Google Sheets API widget.



Enable it.



The next step is to choose the credentials tab on the left the drop-down menu and click the CREATE CREDENTIALS button on the top bar.



Choose API key type.



It's all we can make an API call using CURL or any other REST client.

The example below illustrates how to make a GET HTTP API call using a CURL CLI. So, open a terminal and paste the next code, click ENTER

        curl -X GET https://sheets.googleapis.com/v4/spreadsheets/{SpreadsheetID}?key={API_KEY}

Of course that {API_KEY} and {SpreadsheetID} you replace with the already generated API key on the previous step

SpreadsheetID you can find by opening spreadsheet with which you want to work and copy ID from URL bar
