You can fetch some cells' content as well. For this purpose add sheet name and range optional parameters to your request. An example of such a request is seen below 

curl -X GET 'https://sheets.googleapis.com/v4/spreadsheets/{SPREADSHEET_ID}/values/{SHEET_NAME}!{RANGE}?key={API_KEY}'
Take attention this time we use a single quote for a URL it's crucial because there is ! sign in a range parameter so BASH can't treat the URL correctly in this case, to make BASH work with such URL we wrap the whole URL into a single quote.

Write to Google Sheet

You need to add a Bearer access token to your request header to POST some content in a Google spreadsheet. You can get access tokens on OAuth 2.0 Playground. For this purpose pick Google Sheets API on the left container. To do this quickly you can use a Ctrl + F keyboard shortcut. Start typing the sheet in the search bar. When you find the Google Sheets API item click on it, select https://www.googleapis.com/auth/spreadsheets scope and click the Authorize APIs button.



In this step, you need to login into your Google account and allow OAuth 2.0 platform access to your account. When you've successfuly logged in you click on the Exchange authorization code for tokens button.



If everything is good you should get access and refresh tokens
